pypufferblow.federation module#

class pypufferblow.federation.Federation(options: FederationOptions)[source]#

Bases: object

Federation API wrapper for ActivityPub follow and cross-instance direct messages.

These operations are always performed through the authenticated home instance, which acts as the local authority for federation.
